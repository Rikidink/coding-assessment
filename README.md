# Setup instructions

## API/Backend
1. `cd` to the `backend/` directory
2. Run `npm run dev` to seed the db and run the API

## Client/Frontend
1. `cd` to the `frontend/` directory
2. Run `npm run dev`

## Testing
Using `Vitest` to test API request validation
1. `cd` to the `backend/` directory
2. Run `npm run test`

## Packages/external dependencies used
### API
- `better-sqlite3`
- `drizzle-orm`
- `express`
- `zod`

### Client
- `tailwind`
- `TanStack Query`
